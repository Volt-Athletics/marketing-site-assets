# marketing-site-assets
Volt Marketing Site Assets
A place to store static assets for the Volt Athletics Webflow marketing site.
Program preview JSON
Use the volt-athletics:volt-program-preview-json Claude skill to generate or refresh program preview JSON files. These files power the program preview widgets on the marketing site and are named by sport (e.g. soccer.json, track_and_field.json). Update a file in place and push to have it reflected on the site.
